Creek with stress marked:

|------------------------|------------------------|
| [ífa]                  | dog                    |
| [ifocí]                | puppy                  |
| [nâːnáki]              | things                 |
| [sâːsákwa]             | goose                  |
| [aktópa]               | bridge                 |
| [hoktáki]              | women                  |
| [wanayíta]             | to tie                 |
| [awanayíta]            | to tie to              |
| [isiskitocí]           | little drinking vessel |
| [aːtamihóma]           | bonnet (car hood)      |
| [honantaːkí]           | men                    |
| [ilitohtáɬita]         | to cross legs          |

Observations:
- Primary stress usually falls on the second to last syllable 
- Exceptions:
  - If penultimate vowel is short and final vowel is long, stress the final (ifocí, isiskitocí)
  - If penultimate vowel carries a falling tone, it gets the stress (nâːnáki, sâːsákwa)
- Suffixes and prefixes don't affect stress placement on the root (wanayíta, awanayíta)
- In compounds, each member is stressed separately (aːtamihóma)

Problem 2: 

(a) Translations to English:
1. nɤ ʒip ku ne - Do you_{sg} go to sleep? 
2. ati kəmə nirum lapkʰi tʰi ne - Did he see us?
3. tarum kəmə nuʔrum cʰam ran ne - Do they know you_{pl}?
4. nirum kəmə tarum lan ki ne - Do we beat them?  
5. nirum kəmə nɤ cʰam tiʔ ne - Did we know you_{sg}?
6. nirum ka tiʔ ne - Do we go?

(b) Translations to Hakhun:
7. ŋabə nɤ lan tɤʔ ne
8. tarum kəmə ŋa lapkʰi ri ne  
9. ati kəmə nɤ cʰam tuʔ ne
10. nuʔrum ʒip ruʔ ne

Observations:
- Basic word order: Subject - Object - Verb
- Yes/no questions formed with final particle ne
- Subject pronouns: ŋa "I", nɤ "you_{sg}", ati "he", nirum "we", nuʔrum "you_{pl}", tarum "they"  
- kəmə marks the agent in questions
- Verbs:
  ka "go", ʒip "sleep", lapkʰi "see", cʰam "know", lan "beat"
  Tense/aspect marked by suffixes on the verb: 
    -ʔ/-k past, -Ø present, -i future

(c) Translations to Mountain Arapesh:
17. anowip biaruh bøbys
18. biogu atuhu nybat juhurub
19. anowip nybat juhurubys 
20. anowip biabys atugas baigas 

The Mountain Arapesh number system:
- 1-3 use specific numbers: atub "one", biaruh "two", biagas "three"
- 4-5 add biaruh/biagas to anowip "forks": anowip biaruh "four", anowip biagas "five"
- 6-8 use nybat "legs" + biaruh/biagas/anowip: nybat biabys "six", nybat biabys bøbys "seven", anowip biabys bøbys "eight"
- 9-10 use biogu "knots" + number words: biogu nybat atuh "nine", anowip biagas atugas atug "ten"
- 11-25 use biogu atuhu/biabys/biagas/anowip + number words
- 26-30 use araman "man" + number words

Classifiers are used with numbers:
- Generic: bø-, juhuru-  
- Long rigid objects (sugar cane): ara-
- Flat flexible objects (sago bark): baiga- 
- Oval objects (eggs, betel nuts): -bys
