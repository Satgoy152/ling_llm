19. Awon banuuna oon beneen. 
20. Ok anona ambib wedmoona anon ye weng wengambureen.

Key observations:
- SOV word order
- Subject marked by suffix on first verb: 
   -un "I", -uun "she", -een "he", -uub "we", -iib "they", zero "you(sg)"
- Non-final verbs take suffix -a or -e 
- Object marked by suffix on second verb:
   -un "me", -een "it"
- Future tense marked by prefix on second verb: wen- before consonants, weng- before vowels

Key observations:
- The Book Pahlavi script is an abjad, writing mainly consonants. Some letters represent multiple consonants.
- Certain common words are written with Aramaic heterograms (logograms), e.g. MLKTA for bāmišn.
- Word-final consonants are often written with letters that have an inherent -A vowel.
- Silent letters are sometimes added at the end of words, e.g. -DX in ALBA-DX, -c in ZWZN-c.
- Some consonants are written the same despite being pronounced differently, e.g. d and y, g and w.
- Vowels are sometimes indicated by matres lectionis, e.g. A, W, Y.

Rules:
- Duplicate the first CVC- of the root
- If root starts with vowel, gem      inate the consonant after it
- Stress stays on the same vowel 
- Exceptions:
  - roots with identical vowels in first two syllables (bɛbar, letna, ɛrnɔnaw)
  - CaC- reduplicates as CaC- in coast dialect (not CaCa-)

(a) Translations to English:

(b) Order of Nooni week:
Bvutfu, Bvusɔw, Bvuŋka, Bvuŋkaden, Bvumbon, Bvuzhi, Bvuzhiden

(c) Translations to Nooni:

20. me nɔ̂ː fí dìɛ̀msɛ̌n bvúsɔ̀w.
21. cɔ̌ŋ cí yù cɔ̌ŋ kèŋkfǔ lé.
22. me nú ŋgéː yów mutù bvûŋka᷆ːde᷆n.  
23. kwɔ᷆ːn nǔ zhí dìɛ̀msɛ̌n ɛ̀bɛ̀n.
24. dìɛ̀msɛ̌n cí nɔ̂ː yɛ᷅n wɔ̀ ɛ̀bɛ̀n.

Key observations:
- Day names begin with bvú- or bvû-
- Verbs:
  nú - present, nǔ - future, cí - past, tɔ̀ nɔ̂ː - remote past  
  Progressive ("is about to") formed with nú + verb
  Perfect ("just now") formed with cí + verb + lé
- Tense overrides day name, e.g. fɔgò cǐ nú bvúmbòn "yesterday was Bvumbon"
- Day name for "today" depends on the actual day, not relative 
- Words for "yesterday", "today", etc. inflect for noun class
- Object follows the verb
