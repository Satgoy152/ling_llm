1. **tùŋūn² + tàt + nààs = bākūrū bībā ná vɛ̀ rwīīt**  
   Let:
   - \(a = tùŋūn\)
   - \(b = tàt\)
   - \(c = nààs\)
   - \(d = bākūrū bībā\)
   - \(e = rwīīt\)

   Thus:
   \(a^2 + b + c = d \cdot e\)

2. **tàt nààs = bākūrū bītīīmìn ná vɛ̀ ʃāātàt**  
   Let:
   - \(f = bītīīmìn\)
   - \(g = ʃāātàt\)

   Thus:
   \(b \cdot c = f \cdot g\)

3. **tàāmà² + ʃāātàt + gwīnìŋ = bākūrū bīnāās ná vɛ̀ ʃāāgwīnìŋ**  
   Let:
   - \(h = tàāmà\)
   - \(i = gwīnìŋ\)
   - \(j = bīnāās\)
   - \(k = ʃāāgwīnìŋ\)

   Thus:
   \(h^2 + g + i = j \cdot k\)

4. **ʃāātàt gwīnìŋ = ʃāātàt**  
   Thus:
   \(g \cdot i = g\)

5. **rwīīt² + bà + tùŋūn = bākūrū bītūŋūn ná vɛ̀ ʃāāgwīnìŋ**

   - \(l = bà\)

   \(e^2 + l + a = m \cdot k\)

   Thus:
   \(g^2 + c + b = n \cdot c\)

8. **nààs tàt = bākūrū bītūŋūn ná vɛ̀ nààs**
