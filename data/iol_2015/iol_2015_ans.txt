### Problem 3: Coding System
#### Answers:
- **(a) Rules:**
  1. Leave the first letter in place.
  2. Delete `h` and `w`.
  3. Replace all consonant letters with digits:
      - `bpv` (f): `1`
      - `cgjkqs` (xz): `2`
      - `dt`: `3`
      - `l`: `4`
      - `mn`: `5`
      - `r`: `6`
  4. Reduce any sequence of two or more identical digits to a single digit.
  5. Delete all vowels (a, e, i, o, u, y).
  6. Leave only the first three digits or add zeroes on the right to make the code one letter and three digits long.

### Problem 4: Syntax and Word Order
#### Answers:
- **(a)**
  1. `Alayulujba nguyunyu bungmanyani.` — The old woman will find you (sg.).
  2. `Yagu gininya.` — He left you (sg.).
  3. `Janji darrangguwaji.` — The dog doesn’t have a stick.
  4. `Ngirra nya alanga.` — You (sg.) stole the girl.
  5. `Daguma nyinga.` — You (sg.) struck me.
  6. `Dirragbi ga balamurrungunya.` — She jumped with the spear.
